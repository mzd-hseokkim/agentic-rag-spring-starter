## L1 Single (5줄)

- **변경 영역**: `agentic-rag-autoconfigure/src/main/java/kr/co/mz/agenticai/core/autoconfigure/tools/fs/` 하위 신규 — `FileSystemToolCallbackProvider.java`(Spring AI `MethodToolCallbackProvider` 래핑), `FileSystemTools.java`(`@Tool glob/listDir/readFile` 3 메서드 보유), `dto/DirEntry.java` 1개. 테스트는 동일 모듈 `src/test/.../tools/fs/`에 메서드별 1파일씩 3개. 이벤트 발행(MAE-382)과 autoconfigure 와이어링(MAE-380)은 본 태스크 범위 외 — 시그니처상 `WorkspaceSandbox`/`OutputLimits`만 생성자 주입 후크로 열어둔다.
- **핵심 결정**: (1) `FileSystemTools`는 일반 Java 클래스로 `@Tool` 어노테이션 메서드만 노출 — Spring `@Component` 아님(autoconfigure에서 `MethodToolCallbackProvider.builder().toolObjects(...)`로 등록, MAE-380 책임). (2) tool name은 namespace 충돌 방지를 위해 `fs_glob`/`fs_listDir`/`fs_readFile` 채택(Open Question P3 해결) — MCP 표기와 일치. (3) `glob`은 `FileSystems.getDefault().getPathMatcher("glob:" + pattern)` + `Files.walk` + mtime 내림차순(`Files.getLastModifiedTime`), `headLimit` default = `OutputLimits.maxResults`(50). (4) `listDir`은 `Files.list`(recursive=true면 `Files.walk(maxDepth=Integer.MAX_VALUE)`) → `DirEntry(name, type FILE/DIR/SYMLINK, size, mtime epochMillis)` List. (5) `readFile`은 `Files.lines` stream으로 `skip(offset)` `limit(n)` → `String.format("%6d\t%s", lineNo, line)` prefix. 모든 경로 인자는 진입 직후 `sandbox.resolve(rawPath)` 호출, 반환 `Path`만 사용. (6) 출력은 `OutputLimits.apply(list)` 또는 `OutputLimits.applyString(content)` 헬퍼로 절단 — 절단 시 마지막 줄 `(truncated, N more)` append.
- **검증**: 단위 테스트 3건 — `glob` 정상 매치 + 0매치 빈 결과 + sandbox 거부(`../`) 예외, `listDir` 비재귀/재귀 + 빈 디렉토리, `readFile` offset/limit slice + 범위 초과 시 빈 결과 + sandbox 거부. 추가로 `MethodToolCallbackProvider`로 감싼 뒤 `ToolCallback.getToolDefinition().name()`이 `fs_glob`/`fs_listDir`/`fs_readFile`로 노출되는지 1건. `./gradlew :agentic-rag-autoconfigure:test`.
- **리스크**: MAE-377의 `WorkspaceSandbox`/`OutputLimits` API 시그니처가 아직 확정 전 — 본 태스크는 sibling 진척 후에 main을 머지받아 import 확정 필요. 우회: MAE-377이 노출한 메서드 시그니처가 다를 경우 `FileSystemTools` 생성자 어댑터에서 흡수. mtime 정렬은 `Files.getLastModifiedTime` 호출이 매치 항목 수만큼 IO 호출이라 큰 디렉토리에서 비용 — `headLimit` 기본 50으로 자연 제한, Phase 2에서 캐싱 검토(현재 금지).
- **롤백**: 신규 패키지 `.../tools/fs/` 통째 삭제 + 추가된 import 없음 → revert는 PR 단위 단순 revert로 가능.
