### Info

- **I1. `RagBaggage.set` deprecated API** (이전 리뷰 동일) — `tracer.createBaggage(...).set(...)` 가 micrometer-tracing 1.4.x에서 deprecated. cross-bridge 호환을 위한 의식적 선택, 주석으로 사유 명시. 수용 가능. micrometer-tracing 1.5.x 도입 시 `BaggageManager` API 마이그레이션 필요.

- **I4. streaming `streamObs.stop()` 이중 호출 잠재성** — `DefaultAgenticRagClient.askStream` (L213, L230). `tail` Flux 정상 경로에서 stop 후 downstream에서 에러가 발생하면 `onErrorResume`이 다시 stop 호출. micrometer Observation은 stop 후 재호출에 대해 idempotent하지 않을 수 있어 일부 구현(SimpleObservation)은 무시되지만 contract상 미정의. `AtomicBoolean` 가드 권장. (이전 리뷰 동일, blocking 아님)

- **I6. correlation-id baggage scope 폐쇄 swallow** — `DefaultAgenticRagClient.askStream` L180/L214/L231에서 `baggageScope.close()` 예외를 silently swallow. baggage close 실패는 trace context leak 가능성이 있어 최소한 log debug는 남기는 것이 안전. (개선 권장, blocker 아님)
