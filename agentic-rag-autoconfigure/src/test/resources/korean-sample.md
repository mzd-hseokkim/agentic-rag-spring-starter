# Agentic RAG 모듈 소개

## 개요

이 모듈은 Spring AI 기반의 에이전틱 RAG 시스템을 제공합니다. 다양한 LLM 프로바이더와 벡터 데이터베이스를 지원하며, 한국어 검색에 최적화되어 있습니다.

## 주요 기능

### 하이브리드 검색

벡터 검색과 BM25 키워드 검색을 결합하여 검색 품질을 높입니다. 두 검색 결과를 RRF 알고리즘으로 융합합니다.

### 멀티 에이전트 오케스트레이션

의도 분석, 검색, 해석, 요약, 검증 등 여러 전문 에이전트가 협력해 질의에 답합니다. 각 에이전트는 독립적으로 교체할 수 있습니다.

### Fact-checking

생성된 답변을 원본 문서와 실시간으로 대조해 환각을 줄입니다. 법령이나 규정처럼 정확성이 중요한 도메인에서 특히 유용합니다.

## 설정 방법

애플리케이션 프로퍼티에서 언어와 청킹 전략을 선택할 수 있습니다. 기본 언어는 한국어이며, Nori 형태소 분석기가 자동으로 적용됩니다.
